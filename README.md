# Tee Writes

Static one-page portfolio/service website for Orugboh Tosan and Tee Writes.

## Stack

- Plain HTML
- Custom CSS
- Vanilla JavaScript
- No framework, build step, backend, CMS, payment, or database

## Run Locally

Serve the project from the repository root so absolute asset paths resolve correctly:

```bash
python3 -m http.server 4173
```

Then open:

```text
http://127.0.0.1:4173/
```
